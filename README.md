分类隐藏 玩家模型 (战场自动恢复正常)

/8频道 键入命令 | 效果说明
--- | ---
屏蔽 | 开启/关闭模组 (默认开启) [enabled]
屏蔽 组队 | 隐藏自己队伍的成员 [hideParty]
屏蔽 公会 | 隐藏自己公会的成员 [hideGuild]
屏蔽 黑单 | 隐藏黑单名单的公会 [hideBlack]
屏蔽 刷新 | 手动刷新显示
屏蔽 状态 | 查询模块状态
